BRDC PMIS Project Brief

Construction Project Management Information System · Version 1.0 · Requirements Frozen

V1 — Active

What is PMIS?

A modern, modular, enterprise-grade Construction Project Management Information System — a Construction Command Center for executing projects day-to-day, monitoring the portfolio, running weekly management reviews, and handing finished assets off to the CMMS.

First optimized for fuel station construction, extensible to hotels, warehouses, commercial buildings, and EV charging sites.

Locked Architecture Decisions

Backend

Supabase (PostgreSQL, Auth, Storage, RLS, Realtime)

Frontend

React + TypeScript + Vite · TailwindCSS + shadcn/ui

Scale

~5 concurrent projects today · designed for 30+

Hosting

GitHub + Vercel
